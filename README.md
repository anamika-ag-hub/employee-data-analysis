# employee-data-analysis-ABC company
Python for Data Science Module End Project

## Overview
This project analyzes employee dataset to understand team distribution, positions, age groups, and salary trends.

## Preprocessing
- Corrected height column (150–180)
- Handled missing salary values
- Standardized column names

## Analysis
- Team distribution and percentage
- Position segregation
- Age group analysis
- Salary expenditure
- Correlation between age and salary

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

- analysis.ipynb
